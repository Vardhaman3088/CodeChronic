Intersection of Two Arrays

Easy


Given two integer arrays nums1 and nums2 , return an array of their intersection.
Each element in the result must be unique and you may return the result in any order.

Note:
In the outcome, each element can or cannot appear as frequently as it does in the two arrays.
The results can be in any order.



Hint : (Brute force)

  • Create a new empty list.
  • For every element of either of the list, traverse all elements and check for elements in the other list.
  • In case element is present, add this element to the new list.


Constraints:


  • 1 <= nums1.length, nums2.length <= 1000
  • 0 <= nums1[i], nums2[i] <= 1000

Sample inputs and outputs :
Input Output
nums1 = [1,2,2,1], nums2 = [2,2] [2]
nums1 = [4,9,5], nums2 = [9,4,9,8,4] [9,4]
